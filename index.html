--- layout: default title: Simple Alpheios Enabled Site Template ---

This is a template that you can use to create a simple GitHub Pages website with the Alpheios Reading Tools embedded to provide reading aids.

Instructions on how to copy this template and setup your own website can be found in the Getting Started guide.

You can add sections of Latin, Greek, Arabic or Persian text and have Alpheios be automatically enabled for double-click (Desktop)/long tap (Mobile)
